# -Employee-Promotion-Analysis
Анализ факторов, влияющих на решения о повышении сотрудников, с использованием python, SQL и Power BI.

Основная таблица: employee_promotion_prediction (https://www.kaggle.com/datasets/rohit8527kmr7518/employee-promtion-prediction)

Инструменты: Python (pandas, seaborn, matplotlib), SQL (EDA, агрегации) и Power BI (dashboard)

Выводы:
- Более высокая производительность значительно увеличивает вероятность повышения
- Отдел инженерии имеет самый высокий уровень продвижения
- Количество выполненных заданий оказывает умеренное влияние на повышение
